GuvenCebimde - Vercel-Ready Demo Web Panel

1) İndirin ve zip'i açın.
2) Vercel'e giriş yapın: https://vercel.com/new
3) "Upload" seçeneğini seçin ve bu zip dosyasını sürükleyip bırakın.
4) Vercel otomatik buildCommand olarak vercel.json içindeki komutu kullanacaktır:
   npm install --legacy-peer-deps && CI=false npm run build
5) Deploy tamamlandığında site adresi size verilecektir (ör: https://guvencabimde.vercel.app)
6) Demo giriş bilgileri: kullanıcı: admin, şifre: 1234

Not: Eğer lokal test yapmak istersen:
 - npm install
 - npm start
 (Node 18 önerilir)
